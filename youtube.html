The YouTube Global Statistics Dashboard is an interactive Excel-based tool that provides a comprehensive analysis of YouTube's global performance. Leveraging Power Query, Pivot Tables, and advanced data visualization techniques, the dashboard highlights key metrics such as top-performing channels, category trends, regional engagement, and revenue patterns. Notable insights include identifying the USA and India as the leading countries in video views and showcasing category dominance in Entertainment and Music. The dashboard also tracks channel creation trends since 2005 and visualizes subscriber growth, providing actionable insights into YouTube's global impact.

This project showcases proficiency in data cleaning, transformation, and visualization using Excel. It includes dual-axis charts, heatmaps, and interactive slicers to enable dynamic filtering and exploration. With lessons learned in Power Query, advanced Excel functions like VLOOKUP, and storytelling through data, this dashboard highlights a user-friendly approach to analyzing global trends. Future enhancements include adding real-time updates and additional statistical metrics for deeper insights.
